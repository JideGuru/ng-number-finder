# ✨ NG Number Finder
A Node module to check what network provider a particular Nigerian Phone number is.


## 🎖 Installing
```
dependencies:
  npm install ng-number-finder
```

## ⚡️ Import
```
const number = require('ng-number-finder');
```

## 🎮 How To Use

```
console.log(number('0813888888'));
```
